## Key Features

### Patient Dashboard
- Patient demographics & diagnosis
- Admission & discharge tracking
- Patient status with **conditional formatting**
- Medicine consumption by month/day
- Top medicines using **bar charts and ranking**
- Billing breakdown using **stacked/category charts**
- Total bill and medicine quantity calculated with **DAX measures**

### Doctor Dashboard
- Doctor profile, specialization, department
- Appointment tracking & patient spend
- Doctor fees & commission earned
- **What-If Parameter** for interactive commission calculation
- Patient billing table with totals/subtotals

### Power BI Features Used
- **DAX Measures** for KPIs
- **Slicers** for dynamic filtering
- **Custom Tooltips**
- **Card, Gauge, Table & Matrix visuals**
- **Cross-filtering and Drill-through**
- **Data Modeling & Relationships**

---

## Data Modeling & DAX

- Clean data model using **fact and dimension tables**
- Reusable **DAX measures**: Total Bill, Medicine Quantity, Patient Spend, Doctor Commission
- Optimized visuals for **dashboard performance**
